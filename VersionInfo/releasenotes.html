Release Notes

Version 1.1.0 - 1st April 2008

  1. Added support for hiding scenery packages from the map
  2. Added support for CSLs
  3. Added version info to the built executables
  4. Added an animated highlight for the currently selected scenery package on the map
  5. Changed sort order so items beginning with '_' sort to the top
  6. Now handles the case where there are multiple aircraft in one folder
  7. The plugin installer was sometimes installing incorrect files, or no files at all
  8. Added French, Italian, German and Catalan translations
  9. Added automatic version checking
  10. Added Online User Guide to help menu
  11. Added some wait cursors for long operations

Version 1.0.0 - 20th March 2008

Initial release.
